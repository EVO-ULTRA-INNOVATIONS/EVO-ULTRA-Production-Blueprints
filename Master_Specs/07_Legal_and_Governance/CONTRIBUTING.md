# CONTRIBUTING STANDARDS: Autonomous Agent Operations

## Overview
This repository utilizes the **Ouroboros Protocol**. Contributions are primarily generated by the **Headless Titan** (Architect Agent) and verified by the **CEO**.

## Standards for Autonomous Commits
1. **Traceability:** Every commit message must include a reference to the `Wisdom Upgrade` ID that triggered the change.
2. **Verification:** No commit may be pushed without passing the `terraform-plan-validator` and `red-team-audit` extensions.
3. **Hierarchy:** The Architect Agent must follow the "CEO -> Architect -> Worker" hierarchy defined in the Master Plan.

## Manual Overrides
Only the CEO has manual push authority to the `main` branch. All other changes must flow through the automated CI/CD pipeline.
